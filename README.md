## Instructions

Prerequisites
node installed

- to run the web app

```sh
$ git clone https://github.com/GiadaSimonetti/react-calendar
$ cd react-calendar
$ npm install
$ npm start
```
Your browser should open on localhost:3000, if it doesn't go directly to localhost:3000.
